# Snek
# Overview:

This is a simple game of the arcade classic "Snake" built using Python and Pygame. I built this as a way to practice using Pygame.

[Software Demo Video](https://youtu.be/dRszg6eeapY)

# Development Environment

* Visual Studio Code
* Python 3.10.2
* Git / GitHub
